## About

The Commons working group serves as a bridge between CNCF TAG Security and other bodies within the Linux Foundation, particularly the Open Source Security Foundation (OpenSSF). Its goal is to reduce duplication of security efforts and ensure alignment across the ecosystem.

## Goals

- Create a bridge for knowledge sharing between TAG Security and other Linux Foundation bodies
- Seek opportunities to mitigate duplication of security efforts across foundations
- Ensure hygiene recommendations for CNCF projects align closely with corresponding OpenSSF recommendations

## Scope

- Coordinate discussion between contributors from TAG Security and OpenSSF
- Highlight opportunities for TAG Security to contribute to codebases, standards, and publications that benefit the CNCF ecosystem

## Key Deliverables

1. Aid in the creation of a universal open source project security baseline (in collaboration with OpenSSF)
2. Contribute to the development of evaluation probes for Linux Foundation and CNCF projects against the universal security baseline
